# **App Name**: VoteVault

## Core Features:

- Secure CSV Upload: Enable users to securely upload four types of CSV files: ElectionResults, Elections, VoterHistory, and one additional user-defined type.
- File Type Validation: Validate uploaded files to ensure they are in the correct CSV format.
- Direct Cloud Storage Transfer: Directly transmit uploaded files to a secure Google Cloud Storage bucket.
- Upload Confirmation: Provide confirmation messages upon successful file uploads to Google Cloud Storage.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to convey trust and professionalism, reminiscent of secure data environments.
- Background color: Light Grey (#F0F4F8), creating a clean and neutral backdrop to emphasize data clarity.
- Accent color: Steel Blue (#4A6EA3) for interactive elements and key actions.
- Body and headline font: 'Inter' sans-serif for a clean, modern and highly readable user interface.
- Use flat, minimalist icons in shades of blue and grey to represent data and security.
- Maintain a clean and structured layout for data clarity, ensuring a user-friendly experience for managing election data.
- Subtle transitions and animations on data uploads to give the user a dynamic and seamless experience.
